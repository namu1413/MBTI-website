16가지 개발자 유형

당신은 어떤 유형인가요? 테스트를 통해 정확한 유형을 알아보세요.

🏗️
ARCH
아키텍처 설계자
코드 짜기 전에 설계도부터 그리는 타입. 다이어그램 없으면 불안함.
🔥
HACK
핵앤슬래시 코더
일단 돌아가면 됨. 리팩토링? 나중에 할게. 항상 나중.
🔬
PERF
퍼포먼스 집착자
O(n²)을 O(n log n)으로 바꾸는 데 이틀. 실제 차이는 0.001초.
📚
DOCS
문서화 신봉자
코드보다 README가 더 긴 사람. 주석 없는 코드는 죄악.
🎨
UI
UI 퍼펙셔니스트
버튼이 1px 어긋나면 잠 못 잠. 디자이너보다 예민함.
🤖
AUTO
자동화 광신도
반복 작업 3번이면 스크립트 짬. 자동화 스크립트 만드는 데 3일.
🛡️
SEC
보안 편집증
모든 입력값은 공격. 모든 API는 취약점. 모두가 해커.
☁️
DEVOPS
인프라 구루
코드보다 yaml 파일이 많음. 쿠버네티스 없으면 불안.
